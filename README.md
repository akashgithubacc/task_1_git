Client 
Client uses 
Material UI to make the frontend part seamless.
Axios library is used to make post request onto the backend which is listening for requests on port : 5000

Backend
Uses Express js and Node js to set up the server 
and Mongodb and mongoose for setting up the database.

natural npm package is used to create tokens, generate bigrams.

For making comparision in the 2 bigrams, formula used is :

similarity = (intersection of 2 Bigrams)/(Union of 2 Bigrams).
